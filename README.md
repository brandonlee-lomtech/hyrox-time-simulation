# HYROX Time Simulation

A web-based race strategy simulator for HYROX. Enter your target finish time and division, and the app distributes the time across all 8 runs, 8 workout stations, and RoxZone transitions based on real 2026 season global race data.

## Features

- 10 divisions supported (Pro, Open, Doubles, Relay)
- Proportional time distribution algorithm with run fatigue curve
- Performance level classification (Elite / Advanced / Strong / Average / Beginner)
- Donut chart, run split visualisation, station breakdown table, and detail cards
- Multi-language support (English, Korean, Myanmar)

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

Vite, React, TypeScript, Tailwind CSS
